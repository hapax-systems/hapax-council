## Context

The SDLC methodology explication page was identified as a nice-to-have
for the HN launch. Publishing it strengthens trust framing by showing
the governance, axiom enforcement, and quality gates behind Hapax's
development process.

## Rationale for Deferral

1. **Content exists but is not public-ready.** The AVSDLC methodology
   docs (`docs/methodology/`) document internal evidence contracts, not
   a public-facing explanation of the development methodology.

2. **Publication requires operator action.** The publication bus gates
   require operator review for new public surfaces. This cannot be
   completed autonomously.

3. **HN launch is not blocked.** The submission can proceed without the
   methodology page. The weblog and README already communicate the
   system's nature.
