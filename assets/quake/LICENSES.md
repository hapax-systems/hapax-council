# Screwm-Quake Asset Licenses

Audit date: 2026-05-24.

All assets used in the Screwm tower are freely licensed or project-authored.
No proprietary id Software, Bethesda, Microsoft, or original Quake commercial
data is required or vendored in this repository.

## LibreQuake v0.09-beta

- Source: https://github.com/lavenderdotpet/LibreQuake
- Upstream release observed: v0.09-beta
- License: BSD for LibreQuake art/media assets
- Usage: Base game data (`pak0.pak`, `pak1.pak`) installed outside this
  repository under `~/.darkplaces/id1/`. DarkPlaces uses this free-content
  replacement so the Screwm migration does not depend on original Quake data.
- Local repository status: not vendored under `assets/quake/`.

## Project-Authored Generated Assets

These assets are generated or authored inside the hapax-council repository and
are licensed under the repository's license unless a later asset-specific
license file supersedes this note.

- `assets/quake/maps/screwm.wad` — WAD2 texture package generated by
  `scripts/generate-screwm-wad.py`; includes semantic tower textures, 36 ward
  identity panel textures, drift-line textures, and six camera/source portal
  textures.
- `assets/quake/maps/screwm*.map`, `*.bsp`, `*.lit`, `*.prt`,
  `*.content.json`, `*.texinfo.json` — procedural tower geometry and compiler
  outputs generated by `scripts/generate-screwm-map.py` and the Quake map
  toolchain.
- `assets/quake/models/aoa.mdl` — procedurally generated AoA/tetrix anchor
  with attendant sphere generated by `scripts/generate-aoa-mdl.py`.
- `assets/quake/sound/ambient/*.ogg` — project-authored/generated ambient zone
  loops for the five Screwm semantic levels.
- `assets/quake/qc/progs.dat`, `assets/quake/csqc/csprogs.dat`, and matching
  `.lno` files — compiled QuakeC/CSQC from source under `assets/quake/qc/` and
  `assets/quake/csqc/`.
- `assets/quake/glsl/combined_crc59807.glsl` and
  `assets/quake/config/*.cfg`/`quake.rc` — project-authored runtime shader and
  configuration files.

## Exclusions

- Original Quake/Bethesda/id Software textures, models, sounds, maps, and PAK
  files are not included.
- OBS/compositor screenshots and `/tmp/screwm-*` witness captures are runtime
  evidence, not redistributable game assets.
